# Smart Transportation & Fleet Management System

A Java-based console application to manage fleet operations using Object-Oriented Programming concepts.

## Features
- Role-based access (Admin, Dispatcher, Driver)
- Trip assignment and tracking
- Fuel prediction for routes
- Vehicle service alerts
- Maintenance issue reporting
- File-based data storage

## OOP Concepts Used
- Abstraction (User abstract class)
- Inheritance (Admin, Driver, Dispatcher)
- Polymorphism (method overloading, varargs)
- Interfaces (Authenticatable, Maintainable)
- Encapsulation
- Nested Classes
- Exception Handling

## Technologies
- Java
- File Handling (BufferedReader, FileWriter)

## How to Run
1. Clone the repository
2. Open in any Java IDE (IntelliJ / Eclipse)
3. Run FleetPlatform.java
